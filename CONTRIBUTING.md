Para contribuir a este proyecto, sigue estos pasos:

    Realiza un fork del repositorio.
    Clona tu fork en tu máquina local: git clone https://github.com/TuUsuario/Inby.dev.git
    Crea una nueva rama para tu contribución: git checkout -b nueva-contribucion
    Realiza tus cambios y commits: git commit -m "Descripción de los cambios"
    Sube los cambios a tu repositorio: git push origin nueva-contribucion
    Crea un Pull Request desde tu fork en la interfaz de GitHub.

¡Aprecio tus contribuciones y esfuerzos para mejorar este proyecto!
